# Open-Gloves-Haptic-3D-Scan-Dog-Petting-Simulator
Based on the Original Demo from Danwillm. This demo applies the haptic hotboxes to the meshes of 3D Lidar scans so you can feel the mesh with your LucidVR Haptic Gloves
